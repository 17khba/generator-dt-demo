# generator-dt-demo
前端PC脚手架

## Install
``` javascript
npm i yo -g
npm i generator-dt-demo -g
```

## Usage
``` javascript
yo dt-demo
```

## Features
1. 包含 `scss` 、 `js` 各组件资源
2. 包含 `.editorconfig` 代码风格配置文件
3. 根据命令行交互信息自动生成 `readme.md` 文件（内容包括title、author、createTime、description字段）
4. html文件针对 360 、 搜狗 等浏览器启用极速模式
